1990 WEB 1.0

Fue la primera y en ella solo se podia consumir contenido. Se trataba de informacion a la que se podia acceder, pero sin la posibilidad de inteactuar: era unidireccional.



1991 TIM BERNERS LEE CREA LA WORLD WIDE WEB

Tim Berners Lee, crea World Wide Web mas conocida como "WWW". Propuso un nuevo sistema de "hipertexto" para compartir documentos.



1991 PROTOCOLOS DE INTERNET HTTP Y HTML

World Wide Web fue desarrollado sobre los protocolos existentes TCP e IP. Esta basado en cuatro bloques: formato de texto (HTML), protocolo para el intercambio de documentos (HTML), el cliente que muestre esos documetos y un servidor para dar acceso (HTTPD).



2000 - 2004 WEB 2.0

Permite la intereaccion entre usuarios. Da control total al usuario de manejar su informacion. Usuarios productores como consumidores Web colaboratuva. O'Really Media da por primera ves el termino Web 2.0 en octubre del 2004.
